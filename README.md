# pratice-nav_bar
- 연습용 반응형 헤더 만들기

### 작동원리 

- 창 크기가 꽉 찼을 때는 메뉴가 다 보이고
- 창 크기가 작아졌을 때는 오른쪽 상단 햄버거 모양 클릭시 메뉴가 열림.
  - 이와 같은 설정은 js에 이벤트 리스너로 클릭했을때 메뉴와 아이콘에 toggle() 를 이용하여 액티브가 있다면 빼주고 없다면 추가해주는것으로 구현함.
